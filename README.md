## Objective
To clean and standardize a real-world dataset using Excel by handling missing values,
duplicates, text inconsistencies, and data formatting issues.

## Dataset Used
Netflix Movies and TV Shows dataset (CSV format)

## Tools Used
- Microsoft Excel

## Steps Performed
1. Imported CSV dataset into Excel and saved it as XLSX.
2. Froze header row and applied filters for easy navigation.
3. Identified missing values using filters and conditional formatting.
4. Kept missing values in columns like director and cast to avoid data loss.
5. Created a backup sheet before removing duplicates.
6. Removed duplicate records based on show_id and title.
7. Standardized text fields using TRIM and PROPER functions.
8. Validated date and numeric columns.
9. Created a separate Cleaned_Data sheet.
10. Added Data_Quality_Notes to document data issues.
11. Exported the final cleaned dataset as CSV.

## Files Included
- Raw_Data.xlsx
- Cleaned_dataset.xlsx
- cleaned_dataset.csv

## Outcome
A clean, structured dataset ready for analysis, following professional data
cleaning practices.
